# i.MX6 人脸检测安防 Demo（V1.0 技术方案）

## 1. 项目目标

在 **i.MX6 + Linux 4.9.88** 平台上，构建一个 **完整工程化的人脸检测安防 Demo**，覆盖：

- 摄像头驱动（V4L2）
- 用户态视频采集与处理
- 轻量级 AI 人脸检测
- 嵌入式 UI 显示
- 可维护、可扩展的工程结构

> 本项目目标是 **工程能力验证与系统整合**，而非算法竞赛。以及学习v4l2框架，大型项目的构建达到面试要求等等

---

## 2. 硬件与系统约束

### 2.1 硬件平台
- SoC：i.MX6 系列（Pro / ULL / UL）
- CPU：ARM Cortex-A7 / A9
- 无 NPU / GPU
- 内存：256MB / 512MB

### 2.2 操作系统
- Linux Kernel：**4.9.88**
- 工具链：
  ```text
  arm-buildroot-linux-gnueabihf-gcc
